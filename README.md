🚗 Wallonics – HD & 4K Car Wallpapers

🌐 Live Demo: https://wallonics.netlify.app/

Wallonics is a modern wallpaper platform built for car enthusiasts, offering a stunning collection of HD, 4K, and ultra-HD car wallpapers. It features a futuristic neon-themed UI, smooth user experience, and responsive design for all devices.

✨ Features
🚘 Huge collection of car wallpapers (Sports, Luxury, Modified, etc.)
🎨 Unique cyberpunk-inspired UI with neon design
📱 Fully responsive (Mobile, Tablet, Desktop)
🔍 Search and category-based browsing
🔐 User login system (Google / Phone - planned or integrated)
⬇️ Download wallpapers (for logged-in users)
❤️ Add to Favorites functionality
🎥 Video background for immersive hero section
🎵 Background music integration for better user experience
⚡ Fast loading and smooth animations

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Styling: Custom CSS / (Optional: Tailwind CSS)
Backend (Optional): Firebase / Node.js
Authentication: Google OAuth / Phone OTP (Planned/Integrated)
Deployment: Netlify
